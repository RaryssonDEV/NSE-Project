# NFS-e — Gerador de Nota Fiscal de Serviço
Projeto gerador de Nota Fiscal de Serviço (NFS-e) criado para o exercício prático — autor: **RaryssonDEV**.

## Recursos
- Interface responsiva com Bootstrap 5
- Cálculo automático dos impostos (IRPF, PIS, COFINS, INSS, ISSQN)
- Visualização da nota na própria página
- Geração de PDF via jsPDF
- Dark Mode com alternância
- Botão para copiar JSON com os dados da nota

## Observações
- As alíquotas padrões são apenas sugestão. Confirme com um contador.
- PDF gerado tem layout básico — pode ser melhorado com templates HTML->PDF se necessário.
